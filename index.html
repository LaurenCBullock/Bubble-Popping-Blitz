Overview

Bubble Popping Blitz

Bubble popping blitz is a simple puzzle game in which players will have an allotted number of moves to pop all the bubbles on the board. Upon being popped the bubbles will release droplets in all four cardinal directions that will increase the size of the first bubble they hit, if they are at max size then they too will pop. the game is based on chaining the pops together which optimizes bubbles popped per move. There will also be power-ups that make the game more balanced as some bubbles can stand in the way of large chains, so it is advantageous to obtain them as you go.
